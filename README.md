Система обліку персональних витрат (Варіант 3)
Консольний застосунок на Python для керування фінансами, розроблений як екзаменаційний проєкт. Програма дозволяє відстежувати витрати, групувати їх за категоріями та формувати аналітичні звіти з використанням бази даних PostgreSQL.

Технологічний стек
- Мова програмування: Python 3.x.
- База даних: PostgreSQL.
- Бібліотеки: psycopg2-binary (для роботи з БД), python-dotenv (для безпечного керування паролями).
